# BurgerPlusUC
Название проекта: BurgerPlusUC
Назначение разработанное системы: Для заказов различных продуктов и материалов
Перечень реализованного функционала: 1) Авторизация. Чтобы войти в систему необходимо ввести правильный логин и пароль, а затем ввести капчу. 2) Администратор. Были реализованы такие функции как: добавление, редактирование, снятие блокировки пользователя.
Для создании информационной системы были используемы такие технологии как: Visual Studio 2022 и Microsoft SQL Server Management
Требование к ПО: Windows 10 Pro, процессор Intel(R) Core(TM) i5-8400
Порядок запуска системы: 1) зпустите Visual Studio 2022. 2) Найдите и нажмите на проект BurgerPlusUC
